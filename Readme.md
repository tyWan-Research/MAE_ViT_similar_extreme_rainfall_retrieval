The research in this work is based on two influential vision models and pretraining paradigms:



\- \*\*Vision Transformer (ViT)\*\*, whose model design follows the work proposed by \*\*Google Research\*\*

\- \*\*Masked Autoencoder (MAE)\*\*, whose self-supervised pretraining strategy follows the work proposed by \*\*Facebook Research\*\*



In this study, ViT is used as the backbone model for feature representation, and MAE is adopted as the self-supervised pretraining method.
