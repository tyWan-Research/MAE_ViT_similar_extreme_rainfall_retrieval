The methodology used in this study builds upon two influential advances in machine vision and self-supervised learning:

- **Vision Transformer (ViT)**, originally proposed by **Google Research**, which is adopted as the backbone network for visual feature representation.

- **Masked Autoencoder (MAE)**, originally proposed by **Facebook Research**, which is adopted as the self-supervised pretraining framework.

In this work, these two components are integrated and adapted for the retrieval of historically similar extreme rainfall patterns. Specifically, ViT is used as the feature encoder, while MAE is employed to enhance representation learning through self-supervised pretraining.
